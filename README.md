ESLint-plugin-simple-jsx
===================

Rules for saving simple React Components.

# Installation

```sh
$ npm install eslint-plugin-simple-jsx --save-dev
```

# Configuration
add plugins and enable the rules at `.eslintrc`.
```json
{
  "plugins": [
    "simple-jsx"
  ],
  "rules": {
    "simple-jsx/no-operator": "error"
  }
}
```
